Файл init.php подключается в момент инициализации.
Файл text.php содержит выводимый текст (html между <body> ... </body>).
Файл variables.php некоторые переменные.
Файл functions.php свои функции или php-код.
Файл head.php подключается в секцию HEAD.
Файл header.php подключается в начале секции BODY.
Файл footer.php подключается в конце секции BODY.
